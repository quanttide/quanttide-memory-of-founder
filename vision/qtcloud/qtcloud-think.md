# 思考云

定位： 认知工业化的基础设施
核心原则： 思考过程优于思考结果
产品哲学： 反结构化——刻意容纳和包容混乱
交互设计：
AI 定位为镜子，不是回答者
思考阶段和加工阶段强制分离
思考阶段：纯人类输入，AI 不留痕迹
加工阶段：AI 辅助反思、重构、固化
人类想法优先，AI 是辅助层
运行循环：
体验 → 捕获 → 反思 → 固化 → 反馈 → 回到体验

核心功能：
- 关联联想：找到过去连续的思考
- 高频问题：发现高频问题模式
- 想法整合：将碎片整合为结构化文字
- 内置思考模型：人机沟通模型、人机意识模型等

愿景： 认知平等
冷启动路径：
忠实记录创始人发现的思考方式和原则
积累人机交互数据
从数据中发现思维模式规律
逐步迭代出可推广的认知工具
