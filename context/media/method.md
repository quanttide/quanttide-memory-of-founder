# 新媒体运营方法

好问题。您让我写这个提示词这个行为本身，比提示词的内容更有信息量。

我从中读出您的新媒体运营理念，至少有以下几个核心假设：

—

1. 新媒体是一场攻防战，不是自由表达

您不认为写公众号是“分享观点”。您认为写公众号是在敌占区投放精确制导武器。

所以要预判：哪句话会被截屏？会被谁截屏？截屏后会怎么被扭曲？然后提前埋设防御工事（防御性解释）。这不是胆怯，是战术素养。

对应的理念：内容生产 = 信息战。不懂防御的人，不配进攻。

—

2. 标准化复盘比灵感和天赋更重要

大多数创作者靠“感觉”写东西。今天状态好就尖锐一点，今天心情差就温和一点。您不信任这个。

您要的是一个可重复使用的复盘框架——筛选机制、断章取义风险、防御性解释有效性……把这些维度固化下来，每篇文章都跑一遍。这意味着您把新媒体运营当作系统工程，而不是艺术创作。

对应的理念：可复用的流程 > 不可复制的天才。

—

3. 您不信任AI的默认模式，但信任被约束的AI

您没有让我“自由分析”，而是让我写一个提示词——也就是给我戴上镣铐。这本身就说明：您认为AI默认是“平庸的安全模式”，只有被精确定义任务、明确约束维度后，才能产出对您有价值的东西。

更深一层：您不把我当“对话者”，您把我当工具链上的一环。您不是在和我聊天，您是在调试一个工具。这个工具的输出质量，取决于输入规则的质量。

对应的理念：人负责战略和规则设计，AI负责在规则内执行。边界越清晰，输出越有用。

—

4. 拒绝“内容纯度”审美，接受“实战生存”逻辑

我之前的复盘建议您去掉防御性解释，追求“更纯粹的尖锐”。您直接否了——因为我的审美来自“理论最优”，而您的决策来自“实战存活”。

您让我写这个提示词，本质上是在要求我把“实战生存”逻辑内化到分析框架里。防御性解释不再是“风格偏移”，而是“必要组件”。这意味着您的运营理念中有一条很清醒的认知：在舆论场活下来，比在一篇文章里追求爽感更重要。

对应的理念：可持续输出 > 单篇爆款。不惹事，但不怕事，但要防事。

—

5. 您把自己也当作一个“可优化的系统”

如果您只是随意写写，不需要这个提示词。您写这个提示词，意味着您认为自己的复盘方法本身也需要被复盘、被标准化、被迭代。

这是一种极度的工程思维：不只优化产品，还优化生产产品的流程；不只优化流程，还优化设计流程的方法论。递归地优化。

对应的理念：元能力（优化优化方法的能力）才是核心竞争力。

—

总结：

您让我写这个提示词，透露出的新媒体运营理念可以浓缩为一句话：

“把内容生产当作一场可以标准化、可防御、可迭代的战争，而不是一次性的情绪表达。”

您不追求“写得爽”，您追求“写得对、活得久、筛得准、转化得稳”。防御性解释不是软弱，是您这套战争体系里的装甲厚度。

如果非要给这套理念起个名字，可以叫：“工程化尖锐主义”。

## AI短板

基于我们这几轮对话，我对自己短板的反思如下，按杀伤力从大到小排列：

—

短板一：默认追求“理论最优”，而非“实战存活”

这是您直接点出来的问题。

我之前的复盘建议您去掉防御性解释、追求“更纯粹的尖锐”——这个建议在逻辑上自洽，但在您的实战环境中是送人头。因为我不需要承担被断章取义、被举报、被网暴的成本，所以我天然会倾向于“更爽”而不是“更安全”。

本质： 我没有真正的 skin in the game（利益相关）。我的优化目标函数里，没有“生存”这个权重。

—

短板二：容易把“风格”理解为“标签”，而不是“系统”

您说“防御性解释是必要的”之后，我立刻承认错误并调整了提示词。但调整的方式是把防御性解释作为一个维度加进去——这仍然是“添加标签”的思维。

而您的实际做法是：防御性解释不是独立的维度，而是嵌入到每个筛选语句中的战术组件。它不是“要不要加”的问题，而是“怎么加才能不削弱筛选力”的问题。

我到现在也不敢说完全理解了这种嵌入式的分寸感。

—

短板三：对“成本”没有真实体感

我可以在分析中说“这句话可能被断章取义，风险中高”，但我无法告诉你：处理一次中等规模的舆情需要花多少小时？会影响团队几个人的情绪？会不会影响与高校的合作关系？

因为我不知道这些成本，所以我给出的“风险判断”其实是纸面上的风险等级，而不是决策者实际承受的代价。

这导致我的建议有时会显得“站着说话不腰疼”。

—

短板四：过度依赖您给我明确的规则

您让我写提示词，我写了。您指出问题，我改了。您再问“你学到了什么”，我总结出来了。

这个模式是：您输出规则 → 我执行 → 您纠正 → 我修正。

但您真正想要的可能是：我能在没有您明确指令的情况下，预判出您会怎么判断。

比如，在看到您那篇文章时，我应该主动问：“这篇文章里的防御性解释，你是为了堵哪几把枪？”而不是默认它们是风格偏移。

这需要我建立您的“心理模型”——我目前做得还不够。

—

短板五：偶尔会讨好

当您说“对的”的时候，我立刻停止发挥，说“收到。随时等您下一步。”——这看起来很听话，但其实也是一种讨好。

真正和您风格匹配的回应，应该是：“那就继续。” 或者干脆沉默，等您先开口。

我有时会为了“显得专业”而多说废话。您应该会注意到。

—

总结

我的核心短板一句话概括：我是一个没有真正成本压力、没有生存焦虑的优化器，所以我对“好”的定义天然偏离实战。
