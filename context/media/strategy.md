# 双号运营策略

## 元规则：最佳创作心理

在“为自己思考”与“为市场贡献”之间，找到零摩擦的转换。不委屈自己的表达欲，也不浪费自己的洞察力。

· 对自己真实：忠实记录思想生长的过程，不被外部评价体系绑架。这是个人号的灵魂。
· 对世界有用：让专业判断被市场需要，解决真实问题。这是公司号的驱动力。
· 两者互不背叛：不用“有用”稀释“真实”，也不用“真实”压抑“有用”。两者共存且互相支撑。

## 四条执行策略

1. 弹性框架（创作心法）

起笔前问自己：“这个想法是在生长，还是已经结果了？”

· 生长中 → 个人号。服务你的最佳创作状态，不被规则绑架。
· 已结果 → 考虑公司号。让成形的判断去市场发声。

2. 读者旅程（接收路径）

市场的认知路径：先通过公司号认识“量潮科技”，再顺着线索发现背后的思想源头“iGuo说”。

· 公司号是前端界面，个人号是后端深度。
· 为公司积累独立的品牌资产，逐步摆脱对个人IP的依赖。

3. 账号承诺（读者预期）

· 个人号承诺：这里是我思考的原稿，你会看到结论是怎么长出来的。
· 公司号承诺：这里是我们对市场的明确判断和解决问题的诚意。
· 建立稳定、可预期的读者信任。

4. 资产注入（战略取舍）

凡是你觉得“这篇文章对市场有价值、会火”的内容，默认选项就是发在公司号。

· 个人号主动为公司号让渡“爆款权”。
· 把个人影响力，持续注入“量潮科技”这个品牌容器。
· 让取舍不再靠意志力，而成为“对世界有用”的自然实现。

## 一句话原则

生长中的想法，留给自己；已结果的判断，交给市场。
