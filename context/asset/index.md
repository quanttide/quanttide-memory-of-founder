# 领域定义

嗯，关于数字资产这个领域的定义，目前有两个方向。一个方向是继续保持它作为数字资产的这个核心功能，然后中文名也都以它为核心。领域名就叫数字资产，然后呢，云的名称就叫资产云或者叫数字资产云。

嗯，另外一个方向呢，强调它作为统一的一种资产管理的建模方式，只不过是数字资产优先的，然后就强调这种我们特定的时间，然后，那么在这种情况之下的话，就是我们其实是以数字资产作为一个钩子，把所有的无形资产甚至把有形资产给无形资产化，就是建立元数据的方式，来建立一种统一的资产管理的模型，就是对业务活动产生资产的管理。

然后英文名的话，暂时我是不太算动了，就叫 asset management. 然后呢，教程如果需要分离的话，可以再分离。那个就是不分裂也行，不分离就建建立一个统一的框架，就是资产管理统一的框架。然后呢，把它当成一个单独的学科来管理，就它不是财务管理的一个部分，而是一个独立的学科，就是要管理业务活动。

保暂时保持这种中英文分离的方式其实也没有什么问题，对未来可以再逐渐地改，因为整个体系目前已经自动化到了。就比较支持大范围的调整，并且我们有这个资产云，到时候可以自己去管理自己
