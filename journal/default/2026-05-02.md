# 2026-05-02

不确定做游戏这件事情的帮助是什么。我觉得我更像是在不断开新项目从不同项目里练习平台如何做。虽然说游戏已经是一个比较上层的产出，但是还是感觉哪里不对。

我正在处在一种微妙的情景中。一方面，觉得做游戏很快乐，也认识到了一些可能性。另一方面，又觉得我想要的一种更加自然的表达还没有到位。就是那种只要做任何一个事情都可以自动提取元方法的还没有到位，还是需要手动。为了减少认知负担，我已经停止了实时处理。

我觉得我当前这个阶段搭的这种以游戏和小说驱动的模型，已经逐渐开始适合实际情况了。但是还差点什么。

我感觉我总是在绕弯子。一方面想要快速出 HTML 原型。另一面又害怕出太快没法整合。然后又在建模。

我发现我已经掌握了一个新技能，就是什么时候要丢弃 HTML 原型的脚手架换新的。比如我在分解指挥部的指挥官和参谋人员的操作视角。这个过程又容易丢失全貌，也容易在整合的过程中迷失。

建模会担心无法应用，不建模担心无法理解，这里存在一个认知负担需要弥合。目前是手机上建模理解，电脑上操作整合。

今天进步很大。我理清了产品研发流程，感觉生产力又发生了质变。游戏也有进展，感觉自己在做的也是有意义的。

最有意思的指标是自然地驱动了产品迭代而且不觉得费力。这就是创造力溢出了。这种情况反而要注意好自己的休息。


很有意思啊，就是智谱发现我的行为有一定的确定性，就是有共同的模式，就是我在不断的练习如何在不确定的环境之下保持定力，然后就是在这个。过程之中，其中，然后 DeepSeek 说最关键的能力就是觉察，但是不行动。嗯，我觉得这是很重要的一个。
