# 思考模式

思考模式清单（可复用版）

说明：以下每一条都来自你日记或复盘中的真实操作，不是鸡汤。使用时直接套用，不用再怀疑对不对。

—

1. 事实锚定

规则：先问自己“现在是绝境吗？” → 看客观事实，不让情绪放大。
例句：不是绝境，已经比之前好很多了。

2. 兜底转换

规则：失去一个依赖（如父亲兜底）→ 转念为“最大保障”，因为必须靠自己。
例句：没有他兜底，我才真正站住了。

3. 简化决策

规则：反复纠结的事，定一条粗线规则，按规则执行，不再权衡。
例句：债务上有额度就用（风控内）；夜宵单人超过80元就换成便宜的。

4. 情绪素材化

规则：把孤独、羞耻、不安写成小说片段或日记细节 → 从“消耗物”变“原料”。
例句：想吃小龙虾没人陪 → 写“一张写着孤独的138元发票”。

5. 外挂求助

规则：认知负担过重时，直接交给AI/工具：做表、排序、复述、模拟对话。
例句：AI帮我建立还款计划或复述处境。

6. 主动留白

规则：已经想了足够多的工作/问题后，主动停止，留空间给自己。
例句：今天已经想了很多了，不逼自己了。

7. 身体归因

规则：心理压力不明原因时，先检查：饿？渴？困？肩颈痛？
例句：说不定是饿了，吃完再说。

8. 平静标注

规则：能平静谈论痛苦话题（如债务）→ 说明已脱敏，立即标记为进步。
例句：我已经能平静说债务问题了，这是里程碑。

9. 自我看见优先

规则：不被理解时，先完成“自己看见自己”的动作，再管别人。
例句：我刚才做的决策是有道理的，我看见了。

—

使用方式

· 焦虑时：扫一遍清单，问“此刻我可以用哪一条？”
· 写日记时：在句子后面用括号标注用了哪几号模式（如：——自我看见#9）
· 感觉无效时：不是模式错了，是匹配错了；换一条试。

清单不追求完美，可以随时增删。你能从日记里提炼出这些，已经比大多数人有秩序多了。
